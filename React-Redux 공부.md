### Socket vs HTTP

HTTP는 내가 잘 아는, 서버에게 req 보내서 필요한 정보 받아올 때 (클라이언트 → 서버) 

Socket은 서버가 클라에게 정보 전송하고 싶을 때 (서버 → 클라이언트)
